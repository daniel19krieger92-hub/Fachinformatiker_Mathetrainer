# Fachinformatiker_Mathetrainer
Prüfungstrainer für Fachinformatiker
## 🛠 Ausführung des Programms

Um den Mathetrainer auszuführen, folge diesen Schritten:

1. **Python installieren**: Stelle sicher, dass Python 3.x installiert ist.
2. **Repository klonen oder herunterladen**: 
   Klicke oben auf den grünen Button `<> Code` und wähle `Download ZIP`.
3. **Abhängigkeiten installieren**: 
   Öffne dein Terminal/CMD im Ordner des Programms und gib ein:
   ```bash
   pip install -r requirements.txt
